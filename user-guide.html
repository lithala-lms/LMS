1

Course ID Required

A short unique code for the course. Use capital letters and numbers only — no spaces.

  • Good examples: FIRE001, ENV002, CHEM003
  • Bad examples: fire safety, Course 1 (spaces not allowed)
  • This ID is only used internally. Learners never see it.
  • Each course must have a different ID — you cannot use the same one twice.
2

Course Title Required

The main name of the course shown to learners. Examples: Fire Safety Awareness, Environmental Compliance, Chemical Handling.

3

Subtitle

A short description of the type. Use either e-Learning Module or Training Material.

4

Category

Choose the closest matching category from the dropdown. This is used to filter courses on the Courses page.

5

Price Required

The price in South African Rands, including VAT. Enter numbers only — no R symbol. Example: 1500

6

Duration

The estimated time to complete the course. Example: 3 hours, 4.5 hours.

7

Description

2–3 sentences about what the course covers. This is what learners read before purchasing. Write it in plain, professional language.

Example: "This course covers fire prevention principles, extinguisher selection and use, evacuation procedures and legal requirements under the OHS Act."

8

Cover Image

Paste a link to a photo. The easiest way to get a free professional photo:

  • Go to unsplash.com in your browser
  • Search for something relevant (e.g. "fire safety", "workplace training")
  • Click the photo you like
  • Right-click the photo → Copy Image Address
  • Paste that URL into the Image URL field
  • A preview will appear immediately below
9

Learning Outcomes

Type one outcome per line — what the learner will be able to do after completing the course. Aim for 5–8 outcomes.

Example:

  • Identify fire hazards in the workplace
  • Select and operate the correct fire extinguisher
  • Follow the emergency evacuation procedure
✅

WhatsApp Delivery checkbox

If you tick the WhatsApp delivery checkbox, learners will see a 💬 WhatsApp badge on the course card and will be able to activate the course via WhatsApp. Only tick this if you are ready to manually deliver the course via WhatsApp.

Adding Course Modules

Modules are the sections of your course. Each module represents one topic or lesson. The last module should always be the Assessment.

1

Click + Add Module to add a new row

Each row is one module. You can add as many as needed — most courses have 4–8 modules.

2

Type the module title in the text box

Examples: Introduction to Fire Safety, Fire Classes & Extinguishers, Evacuation Procedures

3

Enter the estimated duration

Type how long this module takes. Examples: 20 min, 30 min, 1 hour

4

Set the last module to "Assessment"

Click the dropdown on the right of the last module row and change it from Content to Assessment. The row will turn purple/blue to show it is the assessment module. This is what triggers the quiz at the end.

5

To remove a module

Click the ✕ button on the right of any module row. This removes it from the list immediately.

✅

Recommended module structure

  • Start with an Introduction module
  • Add 3–6 topic modules covering the main content
  • End with a Final Assessment module (set to Assessment type)
📝 Section 6

Adding Quiz Questions

Assessment questions are multiple choice — the learner picks A, B, C or D. You set which answer is correct. Learners need 80% to pass and earn a certificate.

Course Editor — Assessment Questions
📝 Assessment Questions — Learners need 80% to pass
Q1
What class of fire is caused by electrical equipment?
✕
AClass A
BClass B
CClass C ← Correct
DClass D
✅ Correct answer:
C — third option
1

Click + Add Question

A new question block appears. Add at least 5 questions — more questions means a more thorough assessment.

2

Type the question in the text box at the top of each block

Write a clear, unambiguous question. Example: "What class of fire involves electrical equipment?"

3

Fill in all four answer options (A, B, C, D)

Type one option in each box. All four options should be plausible — avoid obviously wrong answers.

4

Set the correct answer using the dropdown

At the bottom of each question block, use the dropdown labelled ✅ Correct answer to select which option is right — A (first), B (second), C (third) or D (fourth).

5

Repeat for all questions

To remove a question, click the ✕ button in the top right of that question block.

📊

Pass Mark

Learners need 80% to pass and earn a certificate. For a 5-question assessment, they must get 4 out of 5 correct. For 10 questions, they need 8 out of 10.

📄 Section 7

Attaching a PDF Document

You can attach a PDF to any course — learners can scroll through it inside the course player. Great for workbooks, study guides or facilitator materials.

1

Upload your PDF to Google Drive

Go to drive.google.com → click + New → File upload → select your PDF file.

2

Make the PDF public

Right-click the uploaded PDF in Google Drive → Share → click Change to anyone with the link → click Done.

3

Get the preview link

Click the three dots on the file → Open in new window. Look at the URL bar — it will show something like:

https://drive.google.com/file/d/ABCDEF123/view?usp=sharing

Change the end from /view?usp=sharing to /preview so it looks like:

https://drive.google.com/file/d/ABCDEF123/preview
4

Paste the link into the Course Editor

In the 📄 Course PDF Document section of the Course Editor, paste the preview URL into the text box.

5

Click "👁 Preview PDF" to verify it works

The PDF will open in a viewer right inside the editor. You can scroll through the pages to confirm it loaded correctly. If it shows an error, check that the sharing settings are set to "Anyone with the link".

✏️ Section 8

Editing an Existing Course

You can change the price, description, image, modules or questions of any course at any time.

1

Open the Course Editor and sign in

Go to https://lithala-lms.github.io/LMS/course-editor.html and enter the password.

2

Click the course you want to edit in the left sidebar

All courses appear in the left panel with a thumbnail image, title and price. The one you click will be highlighted in blue.

3

Make your changes in the form on the right

All fields are pre-filled with the current data. Change only the fields you need to update — price, description, image, modules, questions, etc.

4

Click 💾 Save & Publish Course

Your changes take effect immediately on the live Courses page. Learners who are currently enrolled will also see the updated version.

⚠️

You cannot change the Course ID

Once a course is saved, its ID is locked. The ID is what links learner progress and purchases to the course. If you need a completely different course, create a new one with a new ID instead of trying to repurpose an existing one.

🗑️ Section 9

Deleting a Course

Deleting a course removes it from the public Courses page. Existing learner data for that course is not deleted.

1

Open the course you want to delete

Click the course in the left sidebar to open it in the editor.

2

Scroll to the bottom and click 🗑 Delete This Course

A confirmation message will appear asking you to confirm. Click OK to delete.

3

The course is removed immediately

It disappears from the public Courses page straight away. The course editor sidebar also updates.

❌

Deletion cannot be undone

Once you delete a course from the Course Editor, it cannot be recovered. If you just want to hide a course temporarily, consider changing the price to a very high number or renaming it with "INACTIVE" in the title rather than deleting it.

🏆 Section 12

Certificates

When a learner passes with 80% or more, a certificate is automatically generated and delivered in three ways.

1

Email — sent to the learner instantly

An email with the certificate number, score and a download link is sent to the learner's email address the moment they pass.

2

Email — sent to both admin addresses

Both info@lithala.com and lithalalms@gmail.com receive a notification with the full result details and certificate number.

3

Dashboard — downloadable PDF

In the learner's dashboard under Certificates, they can click ⬇ Download PDF to open the full printable certificate in a new tab, which they can save as a PDF using their browser's Print function.

4

LinkedIn sharing

The learner can click 💼 LinkedIn from their dashboard to add the certificate directly to their LinkedIn profile's Licenses & Certifications section.

🔢

Certificate numbering

Certificates are automatically numbered in the format LTH-YEAR-XXXX, for example LTH-2025-4782. This number is recorded in the Google Sheet Results tab and in both admin emails. You can use it to verify a certificate if an employer or auditor contacts you.

🔧 Section 14

Troubleshooting

Common issues and how to fix them.

!

"I saved a course but it's not showing on the Courses page"

Try refreshing the Courses page (Ctrl + R or ⌘ R). If it still doesn't appear, open the browser console (F12) and check for errors. Also verify the Course ID has no spaces or special characters.

!

"I'm not receiving result emails"

Check your spam/junk folder first. The emails come from the Google Apps Script associated with your Google Sheet. Make sure the Apps Script has been deployed and re-deployed after any changes to google-apps-script.js.

!

"The PDF is not loading in the viewer"

Check that the Google Drive file sharing is set to "Anyone with the link can view" — not restricted. Also ensure you are using the /preview link format, not the /view format.

!

"The Course Editor password is not working"

The password is set in the file lms-config.js on GitHub. Check that the file has been saved and uploaded correctly. The current default password is LithalaAdmin2025.

!

"The Google Sheet is empty"

The sheet only populates when real learners register and purchase courses. The Apps Script also needs to be deployed correctly. Open the Apps Script editor and check the deployment is still active under Deploy → Manage Deployments.

!

"A learner says they are not getting their certificate"

Check the Results tab in the Google Sheet for their email address. Confirm the Passed column shows Yes. If it does, the certificate was issued — ask them to check their spam folder and also look in their Dashboard under the Certificates tab.
