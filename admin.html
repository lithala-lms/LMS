📋 How to Edit Configuration

Open lms-config.js in a text editor (or on GitHub) and update these values:

// In lms-config.js:
SCRIPT_URL: 'your-apps-script-url',
GOOGLE_CLIENT_ID: 'your-google-client-id',
ADMIN_PASSWORD: 'your-new-password',
PAYFAST.MERCHANT_ID: 'your-payfast-id',
PAYFAST.SANDBOX: false, // Set false to go live
